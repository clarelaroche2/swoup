# swoup

swoup website — live at [swoup-usa.com](https://swoup-usa.com)

## Deployment

This site is hosted on **GitHub Pages** and deploys automatically.

**When are changes reflected online?**

1. Merge your changes into the `main` branch.
2. The [Deploy to GitHub Pages](.github/workflows/deploy.yml) workflow runs automatically (usually takes **1–2 minutes**).
3. Once the workflow completes, your changes are live at [swoup-usa.com](https://swoup-usa.com).

You can track deployment status in the [Actions tab](https://github.com/clarelaroche2/swoup/actions/workflows/deploy.yml).
